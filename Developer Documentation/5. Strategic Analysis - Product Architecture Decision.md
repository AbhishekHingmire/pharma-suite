## The Problem Statement

**Current State**: Pharma-Suite is an **inventory management system** for pharmaceutical distributors.

**User Request**: Add comprehensive employee management with attendance tracking.

**The Confusion**: 
- Will this make the app too complex?
- How do inventory workers relate to non-inventory staff (sales reps, delivery)?
- Should these be separate systems?

## Detailed Design: Modular Monolith Architecture

```

      pages/
        Stock.tsx

      components/

      hooks/
        useInventory.ts

      pages/
        Employees.tsx
        Attendance.tsx
        Leave.tsx
        ActivityTimeline.tsx
      components/
        AttendanceCalendar.tsx
        EmployeeCard.tsx
      hooks/
        useAttendance.ts
        useEmployees.ts
      
    integration/               # Bridge between modules
      InventoryToHR.ts         # Link sales to employees
      HRToInventory.ts         # Employee performance in inventory
```

### **4. Data Correlation Strategy**

#### **Automatic Activity Logging**
Every inventory transaction automatically creates an activity log:

```typescript
// When sale is created
const createSale = (saleData) => {
  // 1. Create sale in inventory module
  const sale = await saveSale(saleData);
  
  // 2. If HR module is enabled, auto-log activity
  if (isModuleEnabled('hr')) {
    await logActivity({
      employeeId: currentUser.id,
      type: 'sale',
      title: `Sale created - ${sale.invoiceNo}`,
      saleId: sale.id,
      customerId: sale.customerId,
      amount: sale.total,
      timestamp: new Date()
    });
  }
  
  return sale;
};
```

**Result**: Zero extra work for employees, automatic activity tracking!
